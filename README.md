# First Impression APP
This is an app where you get a pitch to make a good first impression when making new interractions.

## User stories
- Vote for pitch
- New Pitch
- Comment
- Login

## Features
BE able to get content from various categories namely :
- Pick-up Lines
- Interview Pitches
- Business Pitches
- Product Pitches
